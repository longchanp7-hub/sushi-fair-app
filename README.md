# 寿司フェア

スシロー・はま寿司・くら寿司・かっぱ寿司の期間限定フェアを、スマホで一画面にまとめる個人向けPWAです。

## 主な機能
- 「今日の注目」「本日スタート」「もうすぐ終了」を最上部に表示
- 4社の開催中商品数を写真付きカードで表示
- カードをタップすると各社の公式ページへ移動
- 毎日1回、GitHub Actionsで公式公開情報を更新
- 取得失敗時は前回データを残して警告表示

## 公開
GitHub Pages + GitHub Actionsを使用します。PWAとしてGalaxyのホーム画面に追加できます。

## 料金
外部の有料APIは使用していません。
